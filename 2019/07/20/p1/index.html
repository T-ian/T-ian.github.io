C++解法

twoSum1()方法采用的是双重循环遍历法。时间复杂度为O(N2)，实际运行时间约300ms。twoSum2()方法采用的是两遍循环哈希表法，纸面时间复杂度为O(N)，实际运行时间约30ms。涉及到的C++知识有:

  • vector模板

  • map标准库模板

Java解法

思路同上，这里只展示了两遍循环哈希表法。

涉及到的Java知识点有：

  • Map接口
    使用Map接口中的哈希表HashMap。

  • Integer和int的区别
    注意到在程序中，使用Integer而非int来指定键、值类型。简单来说，Integer是int的包装类，也即：int是个基本数据类型，而Integer则是一个类。一个int型值可以直接使用=号赋给Integer对象。当对一个Integer对象和一个int型变量进行关系运算时，Integer和int变量一样。但是不能对两个Integer对象直接进行关系运算。更进一步说，在一个程序中，任何两个Integer对象在使用==来判断相等时，结果都会是false。这是因为，对两个Integer对象使用==所比较的不是它们的值，而是他们的地址。

  • 输出换行问题

Python3解法

涉及到的python语法知识有：

  • 数组List
    在使用List前，需要用from typing import List在文件头导入。数组的定义非常简单，不需要像C++/Java那样主动声明变量类型。len()函数可以返回一个容器对象的长度，因此可用len()来获取List长度。

  • 类
    在python中，类方法和普通函数有个区别，在类方法的定义中，需要在参数列表的首位提供一个额外参数，默认self，其中存放当前对象的地址。但是self并不是python关键字，这里的self可以用其他名称替换。

  • 函数标注
    函数列表中，在形参后使用:来提供函数标注，用于向调用者提供形参的建议类型；或者提供函数的建议返回类型。

  • python字典
    字典是python中哈希表的一种应用，可以用于建立键和值之间的映射关系。

思想

对于数组一类问题，相关的数据结构有顺序表（数组）、链表、哈希表、多元组等。根据题目对空间和时间的不同要求，可以采用不同的数据结构来优化代码，这里使用哈希表降低了时间复杂度。
